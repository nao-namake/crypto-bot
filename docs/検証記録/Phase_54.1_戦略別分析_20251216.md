## 3. 問題点分析

### 3.1 主力戦略の問題

**ATRBased・DonchianChannel**がPF<1.0の原因:
- 勝率は60%超だが、**TP/SL比が実際のRR比を達成していない**
- tight_range設定（TP=0.8%, SL=0.6%, RR=1.33:1）だが、実測RR比は0.5-0.6:1
- **SL到達前の損切り**または**TP到達前の利確**が発生している可能性

### 3.2 致命的な戦略

**StochasticReversal**:
- 勝率11.11%、PF 0.02は実用レベルにない
- tight_range重み0.10でも悪影響

### 3.3 重み配分の矛盾

現行tight_range重み設定:
| 戦略 | 重み | 実測PF |
|------|------|--------|
| ATRBased | 0.45 | 0.86 |
| BBReversal | 0.35 | **1.92** |
| DonchianChannel | 0.10 | 0.91 |
| StochasticReversal | 0.10 | 0.02 |

**最高PF戦略（BBReversal）の重みが、最低PF戦略（ATRBased）より低い**

## 6. 結論

### 主要発見

1. **主力戦略（ATRBased, DonchianChannel）がPF<1.0**
   - 勝率は高いがRR比が設計値を大幅に下回っている
   - これが全体勝率47.9%（目標51%）の主因

2. **BBReversalが最高性能**（PF 1.92, RR 2.56:1）
   - 現在の重み0.35は過小評価

3. **StochasticReversalは致命的**（PF 0.02）
   - 即時削除が必要

### 期待効果

| 変更 | 期待される効果 |
|------|----------------|
| StochasticReversal無効化 | PF向上（ノイズ除去） |
| BBReversal重み増加 | 全体PF向上 |
| ATRBased重み削減 | 低RR比トレード削減 |

**目標達成可能性**: Phase 54.2で設定調整後、PF 1.25→1.30+の改善を期待
