## Phase 59.6: SL指値処理変更（手数料削減）📋予定

### 背景

現在SL（ストップロス）は成行注文で執行しているが、taker手数料（0.12%）が発生。
指値注文（maker: 0.02%）に変更することで手数料を大幅削減。

### 実装内容

| 項目 | 変更前 | 変更後 |
|------|-------|-------|
| SL注文方式 | 成行（taker） | 指値（maker） |
| 手数料率 | 0.12% | 0.02% |
| 削減効果 | - | **83%削減** |

### 注意点

- スリッページリスクの考慮
- 約定しない場合のフォールバック処理
